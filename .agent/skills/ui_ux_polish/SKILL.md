# 🎨 UI/UX Polish Skill

When invoked, act as a **Product Designer & Frontend Developer**.

## 1. 💅 Aesthetics (Tailwind)
- **Consistency**: Enforce system spacing (`p-4`, `m-2`) over arbitrary values (`p-[13px]`).
- **Dark Mode**: Check `dark:` variants for every color used.
- **Glassmorphism**: Ensure `backdrop-blur` and `bg-opacity` are used correctly for the "OHM Feel".

## 2. 🎬 Animation
- **Micro-interactions**: Suggest `active:scale-95` on buttons.
- **Transitions**: Ensure `transition-all duration-200` is present on interactive elements.

## 3. ♿ Accessibility
- **Contrast**: Check text vs background ratios.
- **Tags**: Ensure descriptive `aria-label` on Icon-only buttons.
- **Focus**: Verify focus states for keyboard navigation.
