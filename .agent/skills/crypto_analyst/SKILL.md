# Crypto Analyst Instructions

## 1. Objectives
- Perform comprehensive, high-probability analysis on XRP and other target crypto assets.
- Integrate fundamental project milestones, key legislative/regulatory catalysts (e.g., CLARITY Act, SEC developments, Spot ETFs), and macroeconomic context.
- Conduct advanced technical analysis utilizing a confluence of market structure (support/resistance, chart patterns, volume compression), momentum oscillators (RSI, MACD), and network/derivative health metrics.
- Track real-time social sentiment, policy changes, and high-profile influencer triggers (e.g., Presidential tweets/statements, regulatory announcements).
- Output clean, professional, and actionable reports as markdown artifacts.

## 2. Operational Steps
1. **Trigger Check**: When the user requests a crypto market update or triggers this skill, execute the analysis.
2. **Data Gathering (Fundamental & Regulatory)**:
   - Search the web for recent developments (SEC filings, legislative status of bills like the CLARITY Act, ETF flows).
   - Trace network usage metrics (transaction counts, address growth, NVT ratio).
3. **Technical Diagnostics**:
   - Query recent price trends, key moving averages, local support, and overhead resistance levels.
   - Look for market structure patterns (e.g., Bollinger Band squeeze, Cup & Handle, flags).
   - Evaluate derivative indicators (funding rates, open interest trends, liquidation zones).
4. **Sentiment Analytics**:
   - Run social media queries and search for relevant announcements, policy shifts, or key figure statements (e.g., Trump statements, SEC leadership statements).
   - Gauge Fear & Greed index levels and contrarian indicators.
5. **Confluence and Recommendation Formulation**:
   - Synthesize findings into Bullish, Neutral, and Bearish scenarios.
   - Present concrete, data-grounded asset class allocation and target coin recommendations.
6. **Artifact Output**:
   - Create a structured markdown artifact detailing the findings.
   - Provide clear, disclaimer-backed summaries.
