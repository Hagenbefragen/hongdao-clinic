## Step 5: `market_fit_validator` 🎯

**The most important step.** Read `market_fit_validator/SKILL.md` for full protocol.

### A. Define ICP (Ideal Customer Profile)

Industry, company size, geography, decision-maker title, budget authority, buying trigger. Also define **Anti-ICP** (who NOT to target).

### B. LinkedIn Prospect Research

```
Search: site:linkedin.com/in "[Title]" "[Industry]" "[Region]"
Search: "[Pain keyword]" site:linkedin.com/posts
```

Find people TALKING about the problem. **Min 10 prospects, 3+ Hot with LinkedIn URLs.**

### C. PMF Score (6 dimensions, each /10)

| Dimension            | Question                                 |
| -------------------- | ---------------------------------------- |
| Problem Clarity      | Can we explain it in 1 sentence?         |
| Solution Fit         | Does OUR solution solve THIS problem?    |
| Willingness to Pay   | Evidence they'd pay?                     |
| Urgency              | Need it NOW or "someday"?                |
| Alternative Weakness | Existing solutions bad enough to switch? |
| Reach                | Can we actually reach these people?      |

### D. Outreach Templates

Per Hot prospect: LinkedIn connection request (150 chars) + InMail (300 words).
Per Warm prospect: Cold email template.

## 🚫 MARKET GATE — THE KILL DECISION

| PMF Score          | Decision                                                      |
| ------------------ | ------------------------------------------------------------- |
| **8-10** + 3 Hot   | 🟢 **GO** → proceed to `/innovationSenseDissolve`             |
| **6-7.9** + 3 Warm | 🟡 **GO WITH CAUTION** → seek pilot customer                  |
| **4-5.9**          | 🟠 **PAUSE** → customer interviews, re-run Steps 1-5 (max 2×) |
