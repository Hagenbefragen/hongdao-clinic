How to Use

Update your website directly from your phone:

  1. Scan & Connect: If not connected, scan the QR code above.
  2. Send Prompts: Text any change requests to your WhatsApp (e.g. "Change retreat title to Yunnan Forest Retreat").

  4. Agent Application: The AI coding agent (Antigravity) reads these requests automatically and modifies the website files for you.
